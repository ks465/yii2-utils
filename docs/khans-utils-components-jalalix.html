JalaliX Some additional capability added to khans\utils\components\Jalali Class.

These capabilities require an instance of class to be present. So the JalaliX class is designed to be instantiated. The two key methods of parent class, khans\utils\components\Jalali::date() and khans\utils\components\Jalali::mktime() are disabled. Instead the constructor allows to create objects.
