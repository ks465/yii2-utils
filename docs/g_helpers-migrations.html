KHanMigration ¶

This is a base class for applying the migrations. It contains some methods specifically for MariaDB/MySQL. For testing purposes

  1. User creates all of user tables and models based on KHanS\Utils\helpers\migrations\KHanMigration::$user_models

  3. Identity
