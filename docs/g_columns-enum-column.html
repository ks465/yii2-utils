EnumColumn Class ¶

KHanS\Utils\columns\EnumColumn extends \kartik\grid\EnumColumn only to simplify routine, uniform development. Vertical alignment is set to middle, and horizontal alignment is set to center. This is true for both the header and the content.

filterType is set to kartik\select2\Select2 so the filter element is consistent with other parts.

attribute and enum are required configurations for this widget to work properly.

[
    'class'            => 'KHanS\Utils\columns\EnumColumn',
    'attribute'        => 'status',
    'enum'             => KHanModle::getStatuses(),
],
