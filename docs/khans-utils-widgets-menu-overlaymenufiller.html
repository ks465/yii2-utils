Class khans\utils\widgets\menu\OverlayMenuFiller

All Classes | Properties | Methods
Inheritancekhans\utils\widgets\menu\OverlayMenuFiller » yii\base\BaseObject

Class OverlayMenuFiller reads a CSV file and builds the menu items required for khans\utils\widgets\menu\OverlayMenu.

You do not need to call this class directly, instead you can configure khans\utils\widgets\menu\OverlayMenu to utilize it.

Example 1

$this->menuFiller = new OverlayMenuFiller([
   'csvFileUrl' => 'sample.csv',
   'tabs' => $this->tabs,
]);

Example 2

$this->menuFiller = new OverlayMenuFiller([
   'csvFileUrl' => 'sample.csv',
   'tabs' => $this->tabs,
]);
'''
