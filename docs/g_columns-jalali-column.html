JalaliColumn Class ¶

khans\utils\columns\JalaliColumn extends DataColumn only to simplify routine, uniform development for dates column. Vertical alignment is set to middle, and horizontal alignment is set to center. The width is fixed, and the cell value is calculated using components\Jalali. This is true for both the header and the content.

filterType is set to **** so the filter element is consistent with other parts.

attribute is required configurations for this widget to work properly.

[
    'class'     => 'khans\utils\columns\JalaliColumn',
    'attribute' => 'created_at',
],

JFormat is only optional configurations:

[
    'class'     => 'khans\utils\columns\JalaliColumn',
    'attribute' => 'updated_at',
    'JFormat'   => Jalali::KHAN_SHORT,
],
