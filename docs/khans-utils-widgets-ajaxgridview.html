Class KHanS\Utils\widgets\AjaxGridView

All Classes | Properties | Methods
InheritanceKHanS\Utils\widgets\AjaxGridView » KHanS\Utils\widgets\BaseGridView » kartik\widgets\Widget

Create a uniform grid view for Ajax controllers along with the required Modal widget required for this to work

See also \kartik\grid\GridView Example: `php common\widgets\AjaxGridView::widget([ 'dataProvider' => $dataProvider, 'filterModel' => $searchModel, 'export' => instance_of_ExportMenu, 'columns' => require(__DIR__ . '/_columns.php'), 'createAction' => ['create', 'id' => $itemID], 'createLabel' => 'Create New Item', 'title' => false, 'footer' => false, 'gridIsModal' => true, 'bulkAction' => [ 'action' => 'bulk-delete', 'label' => 'Delete', 'icon' => 'trash', 'class' => 'danger', 'message' => 'Some alert', ], ]) ` Because this requires a modal window, for simplicity it extends Widget and not GridView.
