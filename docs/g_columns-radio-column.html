RadioColumn Class ¶

khans\utils\columns\RadioColumn extends \kartik\grid\RadioColumn only to simplify routine, uniform development. Vertical alignment is set to middle, and horizontal alignment is set to center. This is true for both the header and the content.

name is fixed to selection so the bulk action button can find the selected button.

This column has nothing to configure. If any radio column is required for the model fields, use the parent one.

[
    'class' => 'khans\utils\columns\RadioColumn',
],
